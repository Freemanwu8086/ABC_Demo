# ABC曲谱系统

毕设项目
SpringBoot+MyBatis的ABC记谱法学习网站，集成了paulrosen大佬的abcjs库，在此献上我的膝盖！
2022-02-16 目前已将基本功能全部完工。
后续有任何更新我会及时提交到master上。
由于此项目是本人的毕业设计项目，若要使用其中代码请与我取得联系，最终解释权归本人所有。
